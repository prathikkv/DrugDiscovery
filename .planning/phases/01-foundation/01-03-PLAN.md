       - `PROJECT_SCHEMA` with CREATE TABLE (see RESEARCH.md Project Schema):
         - `project_id TEXT PRIMARY KEY`
         - `name TEXT NOT NULL`
         - `description TEXT`
         - `created_by TEXT NOT NULL`
         - `created_at TEXT NOT NULL`
         - `updated_at TEXT NOT NULL`
         - `status TEXT NOT NULL DEFAULT 'active' CHECK(status IN ('active', 'archived', 'deleted'))`
         - `config_json TEXT NOT NULL DEFAULT '{}'`
       - `init_project_db(conn)` -- execute schema DDL
       - Query functions: `insert_project(conn, ...)`, `get_project(conn, project_id)`, `list_projects(conn, status='active')`, `update_project_status(conn, project_id, status, updated_at)`

    4. **src/project/service.py**:
       - `ProjectService` class:
         - `__init__(self, db_path: Path = None, audit_trail: AuditTrail = None)`:
           - Default db_path from `config.PROJECTS_DB`
           - Default audit_trail creates new `AuditTrail()` if None
           - Initialize schema
         - `create(name, description, created_by, config=None) -> Project`:
           - Generate UUID for project_id
           - Set created_at and updated_at to `datetime.now(timezone.utc).isoformat()`
           - Insert into database
           - Create per-project directory structure at `config.PROJECTS_DIR / project_id /`:
             - `uploads/` -- raw uploaded files
             - `checkpoints/` -- pipeline checkpoint h5ad files
             - `results/` -- analysis outputs
             - `exports/` -- generated reports
           - Call `self.audit_trail.append_record(created_by, "CREATE", "project", project_id, {"name": name})`
           - Return Project dataclass
         - `list(status='active') -> list[Project]`:
           - Query database, return list of Project dataclasses
         - `get(project_id) -> Optional[Project]`:
           - Query by ID, return Project or None
         - `delete(project_id, deleted_by) -> bool`:
           - Soft delete: update status to "deleted" and updated_at
           - Call `self.audit_trail.append_record(deleted_by, "DELETE", "project", project_id)`
           - Return True on success, False if project not found
         - Do NOT physically delete the row or the project directory. Soft delete only.
         - Each method creates its own connection (thread safety pattern from Plan 01)
  </action>
