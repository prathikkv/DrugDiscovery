   For the actual pipeline function, create a wrapper `_run_omics_pipeline(project_id, file_path, tissue_type, target_genes, max_mito)` that:

   - PENDING: `st.info("Pipeline queued...")`
   - RUNNING: `st.progress(status.progress, text=f"Running... {status.progress:.0%}")`
   - COMPLETED: `st.success("Pipeline complete!")`, store `status.result_json` in session_state under pipeline_result key, st.rerun() outside fragment scope
   - FAILED: `st.error(f"Pipeline failed: {status.error_message}")`

6. **Results display** (when pipeline_result exists):
   - Show QC metrics in metric cards (using `metric_card` from styles)
   - If pipeline_report contains UMAP data, display with st.plotly_chart
   - Show DE summary table with st.dataframe if available

   Gate 2 - Annotation Review: `hitl_gate(gate_id=f"{pid}_omics_annotation", gate_title="Cell Type Annotation Review", module="omics", description="Review CellTypist annotations against canonical markers.", data_summary={"Cell Types Found": "...", "Confidence": "..."})`

   Gate 3 - DE Review: `hitl_gate(gate_id=f"{pid}_omics_de", gate_title="Differential Expression Review", module="omics", description="Review DE results before evidence gathering.", data_summary={"DE Genes": "...", "Significant": "..."})`

8. **Navigation**: After all gates approved, show `st.page_link("src/pages/evidence.py", label="Continue to Evidence Explorer", icon=":material/arrow_forward:")`
