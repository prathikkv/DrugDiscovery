- `QUERY_TARGET` -- GraphQL query fetching target info by ensemblId: approvedSymbol, approvedName, biotype, tractability (label, modality, value), knownDrugs (uniqueDrugs count, rows with drug name, mechanismsOfAction, disease name, phase, status)

Method `fetch(self, gene: GeneIdentifiers, disease_context: str | None = None) -> EvidenceResult`:

2. Call `_query(QUERY_TARGET, {"ensemblId": gene.ensembl_id})` to get target info
3. Call `_query(QUERY_ASSOCIATIONS, {"ensemblId": gene.ensembl_id, "size": 25})` to get associations
4. If disease_context provided: filter associations to highlight matching disease (include all but flag relevance)
5. Build data dict: {"target": target_data, "associations": assoc_data, "tractability": tractability_list, "known_drugs": drugs_list, "disease_association_count": count}
6. Return EvidenceResult with confidence=1.0, data=data dict

Wrap all logic in try/except at the top of `fetch()`. On ANY exception: return EvidenceResult with confidence=0.0, error=str(exception), is_fallback=True.

Properties:
- `source_name` -> "pubmed"
- `source_version` -> "entrez"

Constructor: `__init__(self, max_results: int = 10)`. Store max_results.

Method `fetch(self, gene: GeneIdentifiers, disease_context: str | None = None) -> EvidenceResult`:
1. Build search query: `"{gene.canonical_symbol}[Title/Abstract]"`
2. If disease_context: append `AND "{disease_context}"[Title/Abstract]`
3. Append `AND ("last 5 years"[PDat])` for recency
4. Call `_search(query, self.max_results)` to get PMIDs
5. If no PMIDs: return EvidenceResult with confidence=0.3, data={"papers": [], "total_count": 0, "ai_summary": None}, error=None (valid result: no literature)
6. Call `_fetch_abstracts(pmids)` to get paper details
7. Call `_summarize_abstracts(papers_list, gene.canonical_symbol, disease_context)` to generate AI summary of top 10
8. Build data dict: {"papers": papers_list, "total_count": len(papers), "query_used": query, "ai_summary": summary_text}
   Each paper: {pmid, title, abstract, year, journal, authors (first 3)}
9. Set confidence: 1.0 if papers found, 0.3 if no papers (legitimate result)
10. Return EvidenceResult

Wrap all logic in try/except at the top of `fetch()`. On ANY exception: return EvidenceResult with confidence=0.0, error=str(exception), is_fallback=True.

Private method `_search(self, query: str, max_results: int) -> list[str]`:
- Decorated with tenacity retry (3 attempts, exponential backoff on urllib.error.HTTPError, IOError)
- Use Entrez.esearch(db="pubmed", term=query, retmax=max_results, sort="relevance")
- Return list of PMID strings

Private method `_fetch_abstracts(self, pmids: list[str]) -> list[dict]`:
- Decorated with tenacity retry
- Use Entrez.efetch(db="pubmed", id=",".join(pmids), rettype="abstract", retmode="xml")
- Parse XML results via Entrez.read()
- Extract: pmid, title (ArticleTitle), abstract (AbstractText), year, journal (Journal.Title), authors (first 3 from AuthorList)
- Return list of paper dicts

Private method `_summarize_abstracts(self, papers: list[dict], gene_symbol: str, disease_context: str | None) -> str | None`:
- Purpose: Generate an AI summary of the top 10 abstracts per REQ-203
- Uses the LLM provider from `src.execution.llm` (OpenAI-compatible client, same pattern as Phase 4 reasoning engine uses)
- Import: `from src.execution.llm import get_llm_client` (lazy import inside method to avoid hard dependency)
- Build a prompt: "Summarize the following {len(papers)} PubMed abstracts about {gene_symbol}{' in the context of ' + disease_context if disease_context else ''}. Focus on: key findings, therapeutic implications, and emerging research directions. Be concise (3-5 sentences)."
- Concatenate paper titles + abstracts (truncate total to 8000 chars to fit context window)
- Call the LLM with model="gpt-4o-mini" (lightweight, fast, cheap -- configurable via env var LLM_SUMMARY_MODEL)
- On success: return the summary string
- On ANY failure (import error, API error, timeout): log warning and return None (summary is best-effort, must not break the pipeline)
- Wrapped in try/except -- never raises

Method `is_available(self) -> bool`:
- Quick Entrez.einfo(db="pubmed") call
