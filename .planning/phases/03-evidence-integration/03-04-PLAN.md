Create `src/evidence/aggregator.py` implementing `EvidenceAggregator` class:

Constructor: `__init__(self, sources: list[EvidenceSource] = None, cache: EvidenceCache = None, gene_resolver: GeneResolver = None, max_workers: int = 6, timeout: float = 60.0)`.
- If sources is None: import and instantiate all 6 from sources/__init__.py registry
- If cache is None: create default EvidenceCache()
- If gene_resolver is None: create default GeneResolver()

Method `gather(self, gene_symbol: str, disease_context: str | None = None) -> AggregatedEvidence`:
1. **Resolve gene:** `gene_ids = self.gene_resolver.resolve(gene_symbol)` -- returns GeneIdentifiers with canonical symbol, Ensembl ID, UniProt accession
2. **Check cache for each source:**
   ```python
   results: dict[str, EvidenceResult] = {}
   sources_to_fetch: list[EvidenceSource] = []
   for source in self.sources:
       cached = self.cache.get(gene_ids.canonical_symbol, source.source_name)
       if cached is not None:
           results[source.source_name] = cached
       else:
           sources_to_fetch.append(source)
   ```
3. **Parallel fetch uncached sources:**
   ```python
   if sources_to_fetch:
       with ThreadPoolExecutor(max_workers=self.max_workers) as pool:
           futures = {
               pool.submit(source.fetch, gene_ids, disease_context): source
               for source in sources_to_fetch
           }
           for future in as_completed(futures, timeout=self.timeout):
               source = futures[future]
               try:
                   result = future.result()
                   results[source.source_name] = result
                   # Only cache successful results (confidence > 0.0)
                   if result.confidence > 0.0:
                       self.cache.put(gene_ids.canonical_symbol, source.source_name, result)
               except Exception as exc:
                   results[source.source_name] = EvidenceResult(
                       source_name=source.source_name,
                       confidence=0.0,
                       data=None,
                       error=f"Executor error: {exc}",
                       is_fallback=True,
                       fetched_at=time.time(),
                   )
   ```
4. **Build AggregatedEvidence:** Count sources_failed (confidence == 0.0), compute fetched_at as time.time()
5. Return AggregatedEvidence

Method `invalidate_cache(self, gene_symbol: str = None, source_name: str = None) -> int`:
- Delegate to self.cache.invalidate()
