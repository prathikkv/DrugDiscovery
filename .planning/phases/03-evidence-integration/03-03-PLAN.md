Create `src/evidence/sources/clinicaltrials.py` implementing `ClinicalTrialsSource` class:

CRITICAL: Use ONLY the v2 API. The v1 API at `/api/query/` was retired June 2024.

Properties:
- `source_name` -> "clinicaltrials"
- `source_version` -> "v2"

Constants:
- `BASE_URL = "https://clinicaltrials.gov/api/v2/studies"`

Constructor: `__init__(self, max_results: int = 50)`. Store max_results.

Method `fetch(self, gene: GeneIdentifiers, disease_context: str | None = None) -> EvidenceResult`:
1. Build query params:
   - `query.cond`: use disease_context if provided, else use gene.canonical_symbol + " target"
   - `filter.overallStatus`: "RECRUITING,ACTIVE_NOT_RECRUITING,NOT_YET_RECRUITING"
   - `pageSize`: min(max_results, 100)
   - `format`: "json"
2. If disease_context and gene.canonical_symbol: also add `query.intr` = gene.canonical_symbol (per REQ-204: filter by drug names from DGIdb context)
3. Call `_fetch_studies(params)` with pagination support (follow nextPageToken up to max_results)
4. Parse each study: extract nctId, briefTitle, overallStatus, phases, leadSponsor name, conditions list, interventions list
5. Build data dict: {"trials": trials_list, "total_count": len(trials), "query_condition": condition_used, "query_intervention": intervention_used}
6. Set confidence: 1.0 if trials found, 0.5 if no trials (legitimate result), 0.0 on error
7. Return EvidenceResult

Wrap all logic in try/except at the top of `fetch()`. On ANY exception: return EvidenceResult with confidence=0.0, error=str(exception), is_fallback=True.

Private method `_fetch_studies(self, params: dict) -> list[dict]`:
- Decorated with tenacity retry (3 attempts, exponential backoff on ConnectionError/Timeout)
- GET request to BASE_URL with params, timeout=30
- resp.raise_for_status()
- Parse JSON, collect studies, follow nextPageToken pagination
- Return raw study list (parsing happens in fetch())

Method `is_available(self) -> bool`:
- GET BASE_URL with minimal params (pageSize=1, format=json)
- Return True if 200, False otherwise

Create `src/evidence/sources/uniprot.py` implementing `UniProtSource` class:

CRITICAL: Use ONLY the new REST API at rest.uniprot.org. The old endpoint at www.uniprot.org/uniprot is deprecated.

Properties:
- `source_name` -> "uniprot"
- `source_version` -> "2024"

Constants:
- `BASE_URL = "https://rest.uniprot.org/uniprotkb/search"`

Method `fetch(self, gene: GeneIdentifiers, disease_context: str | None = None) -> EvidenceResult`:
1. If `gene.uniprot_accession` is available: query by accession directly (`accession:{accession}`)
2. Otherwise: query by gene name: `(gene_exact:{gene.canonical_symbol}) AND (organism_id:9606) AND (reviewed:true)`
3. Use `fields` parameter to limit response: "accession,gene_names,protein_name,cc_function,cc_subcellular_location,ft_domain,ft_binding,structure_3d,length"
4. Call `_search(query, fields)`
5. Parse response: extract accession, protein_name (recommendedName.fullName.value), function (FUNCTION comments), subcellular_location (SUBCELLULAR LOCATION comments -> subcellularLocations[].location.value), domains (features where type==Domain), has_structure (structure3D), sequence length
6. Build data dict: {"accession", "protein_name", "function" (list), "subcellular_location" (list), "domains" (list), "has_alphafold_structure", "sequence_length", "gene_names"}
7. Set confidence: 1.0 if entry found, 0.0 on error
8. Return EvidenceResult

Wrap all logic in try/except. On ANY exception: return EvidenceResult with confidence=0.0, error=str(exception), is_fallback=True.

Private method `_search(self, query: str, fields: str) -> dict`:
- Decorated with tenacity retry (3 attempts, exponential backoff on ConnectionError/Timeout)
- GET BASE_URL with params: query, fields, format="json", size=1
- resp.raise_for_status()
- Return resp.json()

Method `is_available(self) -> bool`:
- GET BASE_URL with a trivial query (query="accession:P00533", size=1, format=json)

  <action>
Create `src/evidence/sources/chembl.py` implementing `ChEMBLSource` class:

CRITICAL per research pitfall 3: ChEMBL does NOT support gene symbol lookup directly. Must use UniProt accession -> ChEMBL target ID resolution chain.

Properties:
- `source_name` -> "chembl"
- `source_version` -> "34"

Method `fetch(self, gene: GeneIdentifiers, disease_context: str | None = None) -> EvidenceResult`:
1. If `gene.uniprot_accession` is None: return EvidenceResult with confidence=0.0, error="No UniProt accession available for ChEMBL target lookup"
2. Call `_find_target(gene.uniprot_accession)` to get ChEMBL target ID
3. If no target found: return EvidenceResult confidence=0.0, error="Target not found in ChEMBL"
4. Call `_fetch_activities(target_chembl_id)` to get top 50 bioactivities by pChEMBL value
5. Call `_fetch_mechanisms(target_chembl_id)` to get mechanism of action data
6. Build data dict: {"target_chembl_id", "target_type", "activities" (list with molecule_chembl_id, pchembl_value, standard_type, canonical_smiles), "mechanisms" (list), "activity_count" (total, not just top 50), "mean_pchembl" (computed from top 50), "max_pchembl"}
7. Set confidence: 1.0 if activities found, 0.5 if target found but no activities
8. Return EvidenceResult

Wrap all logic in try/except. On ANY exception: return EvidenceResult with confidence=0.0, error=str(exception), is_fallback=True.

Private method `_find_target(self, uniprot_accession: str) -> tuple[str, str] | None`:
- Use `new_client.target.filter(target_components__accession=uniprot_accession)`
- If results: return (target_chembl_id, target_type)
- If empty: return None
- Decorated with tenacity retry (3 attempts, exponential backoff on Exception -- chembl client raises generic exceptions on network issues)

Private method `_fetch_activities(self, target_chembl_id: str) -> list[dict]`:
- Use `new_client.activity.filter(target_chembl_id=target_chembl_id, pchembl_value__isnull=False).only("molecule_chembl_id", "pchembl_value", "standard_type", "standard_value", "standard_units", "canonical_smiles")[:50]`
- Return list of activity dicts
- Decorated with tenacity retry

Private method `_fetch_mechanisms(self, target_chembl_id: str) -> list[dict]`:
- Use `new_client.mechanism.filter(target_chembl_id=target_chembl_id)`
- Return list of mechanism dicts
- Decorated with tenacity retry

Method `is_available(self) -> bool`:
- Try `new_client.target.filter(target_chembl_id="CHEMBL203")[:1]` (EGFR -- known target)
- Return True if results, False otherwise
