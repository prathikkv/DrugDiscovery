**src/scoring/models.py** -- Define these models (follow exact patterns from research):

1. `VerdictLevel(str, Enum)`: GO, CONDITIONAL, NO_GO (value="NO-GO")

2. `SubScore(BaseModel)`:
   - `name: str` -- sub-score identifier (e.g., "gwas_associations")
   - `value: float = Field(ge=0)` -- actual score
   - `max_value: float = Field(gt=0)` -- maximum possible
   - `description: str = ""` -- human-readable explanation
   - `data_source: str = ""` -- which evidence source contributed
   - Property `normalized -> float`: `round(value / max_value, 3)` (0.0 if max is 0)

3. `DimensionScore(BaseModel)`:
   - `name: str` -- dimension identifier (e.g., "genetic_evidence")
   - `score: float = Field(ge=0)` -- total dimension score
   - `max_score: float = Field(gt=0)` -- max possible (15 or 10)
   - `sub_scores: list[SubScore]` -- component sub-scores
   - `data_coverage: float = Field(ge=0.0, le=1.0, default=0.0)` -- fraction of expected data present
   - Validator `score_within_max`: score cannot exceed max_score, round to 2 decimals

4. `WeightConfig(BaseModel)`:
   - 7 float fields with defaults from REQ-402: genetic_evidence=15.0, expression_biology=15.0, druggability=15.0, safety_selectivity=15.0, competitive_landscape=15.0, clinical_translational=15.0, literature_consensus=10.0
   - Method `normalized() -> dict[str, float]`: divides each weight by total sum. If sum is 0, return equal weights (1/7). Always sums to 1.0.
   - Method `as_dict() -> dict[str, float]`: returns raw weights as dict

5. `CompositeScore(BaseModel)`:
   - `score: float = Field(ge=0, le=100)` -- weighted composite
   - `dimension_scores: list[DimensionScore]`
   - `weights: WeightConfig`
   - `formula_version: str = "v1.0"`

6. `Verdict(BaseModel)`:
   - `level: VerdictLevel`
   - `score: float`
   - `dimension_violations: list[str]` -- dimensions below their minimum
   - `forced_conditional: bool = False` -- True when composite >= 75 but violations exist
   - `rationale: str = ""`

7. `ScorecardResult(BaseModel)`:
   - `gene_symbol: str`
   - `disease_context: str | None = None`
   - `composite: CompositeScore`
   - `verdict: Verdict`
   - `evidence_hash: str = ""` -- SHA256 of input evidence
   - `scored_at: str = ""` -- ISO 8601

8. `ComparativeScorecard(BaseModel)`:
   - `scorecards: list[ScorecardResult]`
   - `ranking: list[str]` -- gene symbols sorted by composite desc
   - Validator: 1-20 scorecards allowed
   - Classmethod `from_scorecards(list[ScorecardResult]) -> ComparativeScorecard`: sorts by score desc

**src/scoring/sub_scores.py** -- 24 extractor functions, each mapping raw evidence data to a bounded numeric score. Follow the DIMENSION_EVIDENCE_MAP from research. Each function takes the relevant evidence data dict (or None) and returns a float.

**Genetic Evidence sub-scores (4 functions, max 15 total):**
1. `compute_gwas_subscore(opentargets_data: dict | None, disease_context: str | None) -> float` (0-5): Follows research rules -- 0=no data, 1=any association, 2=disease-relevant, 3=overall_score>=0.5, 4=>=0.7, 5=>=0.9
2. `compute_genetic_association_subscore(opentargets_data: dict | None, disease_context: str | None) -> float` (0-4): Overall association score. Maps OT overall_score: 0=none, 1=>0, 2=>=0.3, 3=>=0.6, 4=>=0.8
3. `compute_causal_evidence_subscore(opentargets_data: dict | None) -> float` (0-3): Mendelian/causal links. Check datatypeScores for genetic_association type. 0=none, 1=any, 2=score>=0.3, 3=score>=0.6
4. `compute_functional_genomics_subscore(opentargets_data: dict | None) -> float` (0-3): Functional genomics datatype score. 0=none, 1=any, 2=score>=0.3, 3=score>=0.6

**Expression Biology sub-scores (4 functions, max 15 total):**
5. `compute_tissue_expression_subscore(uniprot_data: dict | None) -> float` (0-4): From UniProt tissue expression. 0=no data, 1=expressed, 2=multiple tissues, 3=disease-relevant tissue, 4=high expression in relevant tissue
6. `compute_cell_type_specificity_subscore(omics_scores: dict | None) -> float` (0-4): From optional omics tau score. 0=no data/broad, 1=tau>0.3, 2=tau>0.5, 3=tau>0.7, 4=tau>0.85
7. `compute_subcellular_location_subscore(uniprot_data: dict | None) -> float` (0-4): UniProt subcellular_location. 4=cell surface/secreted, 3=membrane, 2=cytoplasm, 1=nucleus, 0=unknown
8. `compute_expression_disease_link_subscore(opentargets_data: dict | None, uniprot_data: dict | None) -> float` (0-3): Differential expression evidence. 0=none, 1=expressed in disease tissue, 2=evidence of dysregulation, 3=strong DE evidence

**Druggability sub-scores (4 functions, max 15 total):**
9. `compute_druggability_class_subscore(dgidb_data: dict | None) -> float` (0-4): DGIdb gene categories. 4=clinically actionable, 3=druggable genome, 2=drug resistant/enzyme, 1=any category, 0=none
10. `compute_existing_compounds_subscore(chembl_data: dict | None) -> float` (0-4): ChEMBL activity count. 0=none, 1=1-5, 2=6-20, 3=21-50, 4=>50 activities
11. `compute_tractability_subscore(opentargets_data: dict | None) -> float` (0-4): OT tractability labels. Score based on small_molecule, antibody, or other_modalities tractability buckets. 0=none, 1-4 based on bucket level
12. `compute_binding_pocket_subscore(chembl_data: dict | None) -> float` (0-3): ChEMBL max_pchembl value. 0=no data, 1=any value, 2=pchembl>=5.0, 3=pchembl>=7.0

**Safety/Selectivity sub-scores (4 functions, max 15 total):**
13. `compute_expression_breadth_subscore(uniprot_data: dict | None) -> float` (0-4, INVERTED -- broad expression = lower safety score): 4=tissue-restricted, 3=few tissues, 2=moderate breadth, 1=broad, 0=ubiquitous/no data
14. `compute_known_safety_signals_subscore(chembl_data: dict | None) -> float` (0-4): Known adverse events from ChEMBL mechanisms. 4=no signals/clean, 3=minor signals, 2=moderate, 1=significant, 0=severe safety flags
15. `compute_essential_gene_risk_subscore(opentargets_data: dict | None) -> float` (0-4): Gene essentiality. 4=not essential, 3=low essentiality, 2=moderate, 1=context-essential, 0=pan-essential
16. `compute_selectivity_potential_subscore(uniprot_data: dict | None) -> float` (0-3): Protein domains uniqueness. 0=no data, 1=common domains, 2=some unique features, 3=unique binding site/domain

**Competitive Landscape sub-scores (4 functions, max 15 total):**
17. `compute_active_trials_subscore(clinicaltrials_data: dict | None) -> float` (0-4): Active/recruiting trials. 0=none, 1=1-2, 2=3-5, 3=6-10, 4=>10
18. `compute_clinical_phase_max_subscore(clinicaltrials_data: dict | None) -> float` (0-4): Highest phase. 0=none, 1=Phase 1, 2=Phase 2, 3=Phase 3, 4=Phase 4/approved
19. `compute_competitor_density_subscore(clinicaltrials_data: dict | None) -> float` (0-4): Unique sponsors. 0=none, 1=1, 2=2-3, 3=4-7, 4=>7
20. `compute_differentiation_potential_subscore(clinicaltrials_data: dict | None) -> float` (0-3): Gap analysis. 3=no competitors (open field), 2=few competitors, 1=moderate, 0=saturated

**Clinical/Translational sub-scores (4 functions, max 15 total):**
21. `compute_clinical_precedent_subscore(opentargets_data: dict | None) -> float` (0-5): Approved drug for target. 0=none, 1=preclinical, 2=Phase 1, 3=Phase 2, 4=Phase 3, 5=approved drug
22. `compute_biomarker_availability_subscore(clinicaltrials_data: dict | None) -> float` (0-4): Measurable endpoints. 0=no data, 1=basic, 2=several endpoints, 3=validated biomarker, 4=companion diagnostic
23. `compute_translational_evidence_subscore(clinicaltrials_data: dict | None, opentargets_data: dict | None) -> float` (0-3): Phase transition success. 0=none, 1=some, 2=good, 3=strong
24. `compute_patient_selection_subscore(clinicaltrials_data: dict | None) -> float` (0-3): Trial design specificity. 0=no data, 1=broad, 2=enriched, 3=biomarker-selected

**Literature Consensus sub-scores (3 functions + penalty, max 10 total):**
25. `compute_publication_volume_subscore(pubmed_data: dict | None) -> float` (0-4): Paper count. 0=none, 1=1-5, 2=6-20, 3=21-100, 4=>100
26. `compute_review_articles_subscore(pubmed_data: dict | None) -> float` (0-3): Reviews/meta-analyses. 0=none, 1=1, 2=2-5, 3=>5
27. `compute_publication_trend_subscore(pubmed_data: dict | None) -> float` (0-3): Year-over-year growth. 0=declining, 1=stable, 2=growing, 3=rapid growth

**Important implementation rules:**
- Every function handles None input gracefully (return 0.0)
- Round every return value to 2 decimal places
- Each function has a docstring documenting the scoring rules
- No I/O, no side effects -- pure computation only
- Access evidence data fields defensively with .get() and default values

**src/scoring/dimensions.py** -- 7 dimension calculator functions. Each takes relevant evidence data dicts and returns a DimensionScore:

1. `score_genetic_evidence(opentargets_data, disease_context) -> DimensionScore` (max 15)
2. `score_expression_biology(uniprot_data, opentargets_data, omics_scores=None) -> DimensionScore` (max 15)
3. `score_druggability(dgidb_data, chembl_data, opentargets_data) -> DimensionScore` (max 15)
4. `score_safety_selectivity(uniprot_data, chembl_data, opentargets_data) -> DimensionScore` (max 15)
5. `score_competitive_landscape(clinicaltrials_data) -> DimensionScore` (max 15)
6. `score_clinical_translational(clinicaltrials_data, opentargets_data) -> DimensionScore` (max 15)
7. `score_literature_consensus(pubmed_data, contradiction_result=None) -> DimensionScore` (max 10)

Each dimension calculator:
- Calls its sub-score extractors from sub_scores.py
- Creates SubScore objects with name, value, max_value, and data_source
- Computes data_coverage as fraction of non-None inputs that had actual data
- Sums sub-scores for total dimension score
- Returns DimensionScore with all metadata

**literature_consensus special handling (REQ-406):**
- Compute base sub-scores (0-10 total)
- If contradiction_result (ReasoningResult from CONTRADICTION mode) has claims with confidence >= 0.7, count them
- Penalty = min(count_of_strong_contradictions, 4)
- Create a "contradictory_evidence" SubScore tracking the penalty
- Final score = max(0.0, base_total - penalty)
