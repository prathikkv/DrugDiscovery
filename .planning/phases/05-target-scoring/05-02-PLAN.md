**src/scoring/verdict.py** -- Two pure functions:

1. `compute_composite(dimension_scores: list[DimensionScore], weights: WeightConfig) -> CompositeScore`:
   - Normalize each dimension: `dim.score / dim.max_score` (0.0 if max_score is 0)
   - Get normalized weights via `weights.normalized()`
   - Weighted sum: `sum(dim_normalized * weight_for_dim) * 100`
   - Round to 1 decimal place: `round(weighted_sum, 1)`
   - Return CompositeScore with all inputs recorded
   - Handle missing data coverage: if a dimension has `data_coverage < 0.3`, assign neutral score (0.5 normalized) instead of the computed value, and re-normalize remaining weights. This prevents missing data from being conflated with negative evidence (research pitfall #2).

2. `determine_verdict(composite: CompositeScore, minimums: dict[str, float] | None = None) -> Verdict`:
   - Use DIMENSION_MINIMUMS from weights.py if minimums is None
   - For each dimension in composite.dimension_scores, check if it has a minimum defined
   - If minimum exists: compute `actual_frac = dim.score / dim.max_score`; if actual_frac < minimum threshold, add to violations list
   - Apply thresholds per REQ-403:
     - `score >= 75.0 and no violations` -> GO
     - `score >= 75.0 and violations` -> CONDITIONAL (forced_conditional=True)
     - `score >= 50.0` -> CONDITIONAL
     - `score < 50.0` -> NO_GO
   - Build rationale string explaining the verdict: include score, threshold, and any violations
   - Return Verdict with all fields populated

**src/scoring/framework.py** -- ScoringFramework class:

```python
class ScoringFramework:
    def __init__(self, weights: WeightConfig | None = None, minimums: dict[str, float] | None = None):
        self.weights = weights or DEFAULT_WEIGHTS
        self.minimums = minimums  # None uses DIMENSION_MINIMUMS default in verdict

    def score_target(
        self,
        evidence: AggregatedEvidence,
        reasoning_results: dict[str, ReasoningResult] | None = None,
        omics_scores: dict | None = None,
    ) -> ScorecardResult:
        """Score a single target gene from evidence and reasoning data."""
```

Implementation:
1. Extract evidence data dicts from `evidence.results` -- for each source_name, get `result.data` (or None if confidence is 0.0 or result missing)
2. Get contradiction result from reasoning_results if available: `reasoning_results.get("contradiction")` or `reasoning_results.get(ReasoningMode.CONTRADICTION.value)`
3. Call all 7 dimension calculators with the appropriate evidence data dicts:
   - `score_genetic_evidence(opentargets_data, evidence.disease_context)`
   - `score_expression_biology(uniprot_data, opentargets_data, omics_scores)`
   - `score_druggability(dgidb_data, chembl_data, opentargets_data)`
   - `score_safety_selectivity(uniprot_data, chembl_data, opentargets_data)`
   - `score_competitive_landscape(clinicaltrials_data)`
   - `score_clinical_translational(clinicaltrials_data, opentargets_data)`
   - `score_literature_consensus(pubmed_data, contradiction_result)`
4. Call `compute_composite(dimension_scores, self.weights)`
5. Call `determine_verdict(composite, self.minimums)`
6. Compute evidence hash: SHA256 of JSON-serialized evidence data (sort_keys=True for determinism)
7. Return ScorecardResult with gene_symbol from evidence.gene.canonical_symbol, disease_context, composite, verdict, evidence_hash, scored_at=ISO8601 now

Also add a convenience function at module level:
```python
def score_target(evidence: AggregatedEvidence, **kwargs) -> ScorecardResult:
    """Convenience: create ScoringFramework and score a target."""
    return ScoringFramework(**kwargs).score_target(evidence)
```
