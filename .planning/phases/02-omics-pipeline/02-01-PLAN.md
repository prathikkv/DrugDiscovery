**src/pipeline/checkpointing.py:** Create checkpoint management functions:

- `save_checkpoint(adata: AnnData, project_dir: Path, stage_name: str) -> Path`: Saves adata to `{project_dir}/checkpoints/{stage_name}.h5ad`. Creates directory if needed. Returns the path.
- `load_checkpoint(project_dir: Path, stage_name: str) -> Optional[AnnData]`: Returns AnnData if checkpoint exists, None otherwise.
- `get_latest_checkpoint(project_dir: Path) -> Optional[tuple[str, Path]]`: Checks for checkpoints in stage order (ingest, ambient_rna, qc, processing, annotation, de, enrichment) and returns (stage_name, path) of the latest one, or None if no checkpoints.
- `save_stage_report(report: dict, project_dir: Path, stage_name: str) -> Path`: Saves report JSON to `{project_dir}/results/{stage_name}_report.json`.
- `load_stage_report(project_dir: Path, stage_name: str) -> Optional[dict]`: Loads report JSON if it exists.

Define STAGE_ORDER as a module-level list: ["ingest", "ambient_rna", "qc", "processing", "annotation", "de", "enrichment"]

**src/pipeline/progress.py:** Create `StageProgressTracker` class:

- `__init__(self, task_manager: TaskManager = None, task_id: str = None)`: Stores references. If either is None, progress updates are silently skipped (allows running pipeline without TaskManager for testing).
- STAGE_WEIGHTS dict mapping stage names to cumulative progress: ingest=0.10, ambient_rna=0.15, qc=0.25, processing=0.50, annotation=0.65, de=0.80, enrichment=0.90, finalize=1.00
- `update(self, stage_name: str, checkpoint_path: str = None)`: Calls task_manager.update_progress(task_id, STAGE_WEIGHTS[stage_name], checkpoint_path) if task_manager and task_id are set.
- `update_substage(self, stage_name: str, fraction: float)`: For intra-stage progress. Interpolates between previous stage weight and current stage weight. E.g., update_substage("qc", 0.5) reports progress = 0.15 + 0.5*(0.25-0.15) = 0.20.
