# Cold DM / Outreach Templates

## Target Profile
- Title: Director/VP/Head of Computational Biology, Target ID, Drug Discovery
- Company: Series B-C biotech (50–500 people), or Big Pharma innovation team
- Where to find: LinkedIn search → "Computational Biology" + "Biotech" + location filter

---

## Template 1 — Demo Offer (Recommended First Message)

**Subject / Opening line:**
> Hi [Name] — I built an AI platform that automates drug target triage. Mind if I show you a 15-minute demo on one of your current targets?

**Full message:**
> Hi [Name],
>
> I noticed you're leading computational biology at [Company] — looks like you're working on [disease area/program if visible on profile].
>
> I recently built BioOrchestrator, an AI platform that takes a gene target and produces a GO/NO-GO recommendation in 15 minutes — aggregating OpenTargets, PubMed, ChEMBL, DGIdb, UniProt, and ClinicalTrials simultaneously, running 5 parallel reasoning modes, and generating a 21 CFR Part 11 compliant audit trail.
>
> I'm offering free 20-minute demos to computational biology teams this month. Happy to run it live on any target you're currently evaluating — no prep needed on your end.
>
> Worth a quick look? I can send a Calendly link.
>
> [Your name]

## Response Handling

**"Interesting, tell me more"**
→ Send Loom link + offer Calendly for 30-min call

**"What does it cost?"**
→ "For early design partners I'm offering a free 2-week pilot on 3 targets. If it adds value, we can talk about a paid engagement. No pressure either way — I just want feedback from people doing real target ID work."

**"We already have tools for this"**
→ "Totally makes sense. The main thing that's different is the GxP audit trail and the AI reasoning synthesis — most tools give you data, this gives you a defensible decision. Happy to show you in 15 minutes if you're ever curious how it compares."

**"Send me more info"**
→ Send the 1-page capability deck (see capability_deck_outline.md) + demo URL

## Target List Sources

1. LinkedIn Sales Navigator (free trial) → search "Computational Biology" + "Biotech" + "Director OR VP OR Head"
2. Y Combinator portfolio (ycombinator.com/companies) → filter by "Biotech" → find comp bio leads
3. Crunchbase → filter Series B-C biotechs → find leadership on LinkedIn
4. STAT News, FierceBiotech job postings → companies actively hiring comp bio = active target ID programs
5. AACR / Bio-IT World attendee lists (often public)
